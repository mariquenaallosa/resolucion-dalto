x) EVENTOS DE MOUSE
    - click -> ocurre con un click
    - dblick -> ocurre con un doble click
    - mouseover -> ocurre cuando el puntero se mueve sobre un elemento o sobre uno de sus hijos
    - mouseout -> ocurre cuando se mueve el puntero fuera de un elemento o de sus hijos secundarios
    ====otros====
    - coontext -> ocurre con un click en el botón derecho en un elemento para abrir un menú contextual 
    - mouseenter -> ocurre cuando el puntero se mueve sobre un elemento (especial para internet explorer)
    - mouseleave -> ocurre cuando el puntero se mueve fuera de un elemento (solamente cuando sale afuera)
    mousedown -> ocurre cuando un usuario apreta un botón del mouse sobre un elemento.
    - mouseup -> ocurre cuando un usuario suelta un botón del mouse sobre un elemento
    - mousemove -> ocurre cuando el puntero se mueve mientras está sobre un elemento


x) EVENTOS DEL TECLADO 
    - keydown -> ocurre cuando una tecla se deja presionada
    - keypress -> ocurre cuando una tecla se presiona y se suelta
    - onkeyup -> ocurre después de que los dos eventos anteriores hayan concluido

X) EVENTOS DE LA INTERFAZ
    - error -> ocurre cuando sucede un error durante la carga de un archivo multimedia
    - load -> ocurre cuando un objeto se ha cargado(funciona con window) 
    - beforeunload -> ocurre antes de que el documento está a punto de descargarse 
    antes de que te vayas del sitio
    - unload -> ocurre una vez que se ha descargado una página
    cuando te vas del sitio
    - resize -> ocurre cuando se cambia el tamaño de la vista del documento
    - scroll -> ocurre cuando se desplaza la barra de desplazamiento de un elemento
    - select -> ocurre después de que el usuario selecciona algún texto de <input> o <textarea>
